Abstract

Storey [1] shed a new light on multiple-testing problem with the definition of positive false discovery rate measure and gave new perspective with direct approach to false discovery rates.

We provide formal introduction to multiple-testing problem and intuitive explanation of Storey‚Äôs direct approach. We demonstrate the duality between Storey‚Äôs direct approach and Benjamini-Hochberg (BH) procedure [2] when using Storey‚Äôs estimate œÄÃÇ0 of the proportion of null hypotheses œÄ0 in BH procedure.

We show that the estimator œÄÃÇ0 of the proportion of null hypotheses œÄ0 plays a key role in multiple-testing problem and that the approach we take doesn‚Äôt matter much. We confirm this using simulations where performances were practically identical for both methods.

We show that Storey‚Äôs estimator œÄÃÇ0 can be very upward biased when distance between distributions under null hypothesis and alternative hypotheses is small. In our simulations this upward bias of œÄÃÇ0 was reduced if we increased the value of the tuning parameter Œª and especially by tuning this Œª parameter using the bootstrap method. However this bootstrap method can result in underestimation of œÄ0, as already explained in [3].

In case of dependence Storey suggested the same approach [4]. We show, this time only by simulation, that dependence can lead to a very high overestimation of false discovery rate.

Keywords: Multiple testing, Multiple comparisons, False discovery rate

1.1 Single-hypothesis testing

Following the scientific method, researchers normally want to establish the truth of a statement by showing that the opposite appears to be false. In general, a hypothesis is a proposed explanation for a phenomenon that one can test. 1.3.3 Storey‚Äôs approach

Storey suggested [1] a modified version of FDR error measure called the positive false discovery rate (pFDR):

Storey suggested in [1] a more direct approach to multiple-testing problem. Rather than fixing level Œ± control of some error measure and then estimating the rejection region (kÃÇ in BH procedure) satisfying level Œ± control (FDR ‚â§ Œ± for BH procedure), we instead fix the rejection region and then estimate some error measure, preferably pFDR.

Important part of Storey‚Äôs direct approach is Storey‚Äôs method of estimating the proportion of true null hypotheses œÄ0 = m0‚àïm. For a good estimator of (p)FDR, we require a good estimator of œÄ0. This problem of estimating œÄ0 is described in [8] and [9]. Storey‚Äôs method of estimating œÄ0 is based on the following idea. Since the m observed p-values contain information about the proportion of true null hypotheses œÄ0, Storey introduces a tuning parameter Œª ‚àà (0,1) and use information in

to estimate œÄ0.

In cases where m0 << m (or equivalently œÄ0 << 1), direct approach using Storey‚Äôs estimators has much more detection power than BH procedure, which takes m instead of an estimate of m0. Equivalently, we can improve detection power of BH procedure by using an estimate of m0, but it is no longer guaranteed to achieve FDR control at the desired level. One advantage of using direct approach instead of BH procedure is that we can estimate pFDR instead of FDR.

8 Conclusions

We illustrate the duality between Storey‚Äôs direct approach and Benjamini-Hochberg (BH) procedure when using Storey‚Äôs estimate œÄÃÇ0 of proportion of null hypotheses œÄ0 in BH procedure. The performances of the two methods are almost identical in our simulations. This demonstrates that the estimator of œÄ0 plays a key role in multiple hypotheses testing. We show that Storey‚Äôs estimator œÄÃÇ0 can be very upward biased when distance between distribution under null hypothesis H0 and distribution under alternative hypotheses H1 is small. In our simulations this upward bias of œÄÃÇ0 is reduced if we increase the value of Œª and especially by tuning the parameter Œª using Storey‚Äôs bootstrap method. However this bootstrap method can result in underestimation of œÄ0. In case of dependence, we show by using simulation that dependence can lead to a very high overestimation of FDR.
